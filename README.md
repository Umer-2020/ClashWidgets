# Clash Widgets
This app is made with the express purpose of creating widgets that are mainly going to help me with keeping track of my base in the game Clash of Clans by Supercell

- You can check out the original game on Google Play Store or Apple App Store under the name _"Clash of Clans"_ by _Supercell_. This is NOT a recreation of the game in any way
- The project is heavily inspired by Clash Ninja the online website that allows one to keep track of their base's progress. Check out the website on https://www.clash.ninja/
- The end goal is for me to have widgets on my phone from which I can quickly reference things like builder time, research time, etc
- I am not going to be making anything that is going to go against the TOS of the game.
